٤

رقم الجواز مكرر — ماذا يعني؟
الجواز موجود في مجموعة أخرى. المشرف يستطيع نقله أو استعادته.
كيف أنقل جواز من مجموعة لأخرى؟
ابحث كمشرف ← اضغط "نقل واستعادة" ← اختر المجموعة الجديدة ← تأكيد.
الجواز محذوف — كيف أستعيده؟
المشرف يرى المحذوفين (فلتر "محذوف"). بالإضافة مجدداً يُستعاد بجميع بياناته.
النظام لا يستجيب عند التشغيل؟
استخدم تشغيل يدوي.bat لرؤية رسائل الخطأ، ثم شارك الخطأ مع الدعم.
Python غير موجودة؟
ثبّت من python.org وتأكد من تفعيل "Add Python to PATH".
كيف أغير كلمة السر؟
المشرف → الإعدادات → المستخدمون → تعديل الحساب → أدخل كلمة السر الجديدة.
